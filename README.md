# Online-Retail-Customer-Behavior-Analysis

## 📌 Overview

This project analyzes customer purchasing behavior using the Online Retail Dataset. The objective is to discover valuable business insights through Data Mining techniques including Classification, Clustering, and Association Rule Mining.

The project provides actionable recommendations for customer segmentation, spending prediction, and product recommendation systems.

---

## 🎯 Objectives

- Analyze customer purchasing behavior.
- Segment customers into meaningful groups.
- Predict customer spending levels.
- Discover product associations for cross-selling.
- Generate business intelligence insights.

## 📊 Dataset Information

Dataset: Online Retail Dataset

Records: 541,909 Transactions

Business Type: UK-based Online Retail Store

Time Period:
December 2010 - December 2011

Source:
UCI Machine Learning Repository / Kaggle

---

## 🔄 Data Preprocessing

- Removed missing values
- Removed duplicate records
- Removed cancelled transactions
- Removed negative quantities and prices
- Created TotalPrice feature
- Extracted Month, Day, Hour features
- Created CustomerRegion feature
- Created SpendingLevel feature

---

## 🤖 Implemented Models

### 1. K-Means Clustering
Customer Segmentation

### 2. Decision Tree
Spending Level Prediction

### 3. K-Nearest Neighbors (KNN)
Classification Comparison

### 4. Logistic Regression
Spending Probability Estimation

### 5. Apriori Algorithm
Market Basket Analysis

---

## 📈 Key Insights

- Peak purchasing hours: 11 AM - 2 PM
- Peak days: Tuesday to Thursday
- UK customers represented the majority of transactions.
- Non-UK customers showed higher spending behavior.
